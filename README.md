

## 🔁 Jenkins Pipeline

The Jenkins pipeline automates:

- Log generation  
- Anomaly detection  
- Artifact storage  
- Build success/failure based on anomalies  

### Run in Jenkins:
- Create new Pipeline Job  
- Add `Jenkinsfile`  
- Click **Build Now**  

---

## 📊 Dashboard Features

- Real-time log streaming  
- Live anomaly trend graphs  
- Log level filtering (INFO / WARNING / ERROR)  
- Metrics (Total logs, errors, anomaly rate)  
- Auto-refresh capability  

---

## 🚨 How Anomaly Detection Works

- Logs are scanned for `"ERROR"` patterns  
- Errors are counted as anomalies  
- If anomalies exist → pipeline fails  
- History is stored for trend analysis  

---

## 📸 Output

- Graphs update dynamically
